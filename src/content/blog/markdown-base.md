# Markdown 引用语法
要创建块引用，请在段落前添加一个 > 符号。<br>
块引用可以包含多个段落。为段落之间的空白行添加一个 > 符号。<br>
块引用可以嵌套。在要嵌套的段落前添加一个 >> 符号。
```
> 来和我缔结契约成为魔法少女吧
>
> 萝莉有三好，身娇体弱易推倒
>
>> 你所热爱的，就是你的生活。
```

>
> 萝莉有三好，身娇体弱易推倒
>
>> 你所热爱的，就是你的生活。

# Markdown 代码语法
要将单词或短语表示为代码，请将其包裹在反引号中。<br>
```
Hello `World`
```
Hello `World` <br>
如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号中。
